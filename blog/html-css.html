What is the difference between Margin, Border, and Padding?

  1. Margin

    2. Definition: The space outside an element's border.
    Purpose: Controls the spacing between the element and other elements around it. Affects how elements are positioned relative to each other.
    Example: Imagine a picture frame. The margin is the space between the frame and the wall.

  2. Border

    3. Definition: The line that surrounds an element.
    Purpose: Provides visual separation between elements. Can be customized with colour, width, and styles.
    Example: The actual frame of the picture frame.

  3. Padding

    4. Definition: The space inside an element's border.
    Purpose: Creates space between the element's content and its border. Can be used to improve readability or visual appeal.
    Example: The space between the picture and the inside edge of the picture frame.

    Summary

    Margin: Controls the space outside the element.
    Border: The line that surrounds the element.
    Padding: Controls the space inside the element, between the content and the border.
